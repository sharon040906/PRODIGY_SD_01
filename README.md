# 🌡️ Temperature Converter (Python)

A simple Python project that converts temperature values between Celsius, Fahrenheit, and Kelvin.

## 🚀 Features
- Convert Celsius to Fahrenheit and Kelvin
- Convert Fahrenheit to Celsius and Kelvin
- Convert Kelvin to Celsius and Fahrenheit
- User-friendly command-line interface

## 🛠️ Technologies Used
- Python

## 📌 How to Run
1. Install Python
2. Run the script
3. Enter the temperature and unit when prompted

## Sample Output
Enter temperature: 25  
Convert to (C/F/K): F  
Output: 77°C


## 💡 Learning Outcomes
- Conditional statements
- Functions
- User input handling

## 🧠 Key Concepts Explained

### 🔹 Tkinter
Tkinter is Python’s built-in library for creating graphical user interfaces (GUI).  
It allows users to interact with applications using windows, buttons, labels, and input fields instead of the command line.

### 🔹 GUI (Graphical User Interface)
A GUI is a visual interface that allows users to interact with software through graphical elements like buttons and text boxes.

### 🔹 Functions
Functions are reusable blocks of code used to perform a specific task, improving code readability and modularity.

### 🔹 Event Handling
Event handling refers to executing code when a user interacts with the interface (e.g., clicking a button).
